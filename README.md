# Responsive Design Maquetacion Mobile First
desarrollo web

El 50% del tráfico en internet se realiza desde dispositivos móviles, convirtiendo una prioridad el desarrollar frontend para smartphones, tablets y laptops.

Se creó un frontend de un sitio web partiendo de su wireframe, analizando su arquitectura se construyó en código cada una de sus partes para que este se adapte a cualquier dispositivo de los usuarios, ya sea Smartphone, tablet o PC.
